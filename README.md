# 山健客 - 山友活動邀約平台 ⛰️

這是一個專為台灣登山愛好者設計的平台，提供活動邀約與報名功能。

### 🏔️ 靈感來源
來自於對大自然的熱愛，曾造訪過：
* **北三錐山** (大雪山森林遊樂區)
* **東卯山大O路線** (谷關)
* **雲嘉五連峰**

### 🛠️ 技術架構
* **前端**: React + TypeScript (Vite)
* **後端/資料庫**: Firebase
* **AI 支援**: Google Gemini (透過 AI Studio 開發)

### 🚀 如何執行
1. 安裝環境：`npm install`
2. 啟動專案：`npm run dev`
